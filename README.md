# Car Rental Management System

A Java-based web application developed to streamline vehicle rental operations. The system allows users to browse available cars, book rentals, and manage reservations while providing administrators with tools to manage vehicles and customer records.

## Features
- User registration and login
- Browse available cars
- Car booking and rental management
- Add, update, and delete vehicle records
- Customer management
- CRUD operations for efficient data handling
- Secure database integration

## Technologies Used
- Java
- Servlets
- JSP
- Hibernate (JPA)
- MySQL
- HTML
- CSS
- MVC Architecture

## Project Architecture
The application follows the MVC (Model-View-Controller) architecture to separate business logic, presentation, and data access layers for better maintainability and scalability.

## Database
MySQL is used as the backend database to store vehicle, customer, and booking information. Hibernate JPA is used for database interaction and object-relational mapping.

## Key Functionalities

- Customer Registration
- Booking Management
- Rental Tracking
- Database Operations using Hibernate
